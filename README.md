## Description
<p>This landing website was developed for an apartment design agency in Kazakhstan where you can order one of the standard layouts of any room from the available catalog and automatically calculate the cost of the design, depending on the selected services. It also shows the general information about the agency, ways of possible cooperation, prices policy, frequently asked questions and other solutions for potential customers.</p>
